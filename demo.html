jQuery Audio Effects

A couple examples of audio effects in action.

jQuery Audio Effects Plugin Tutorial

Written By Nolan R Campbell - http://audioeffects.nrcstudios.info

Including Scripts and CSS files

Add the following scripts to the head section of the page. 

  <script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.6.2/jquery.js"></script>
  <script src="http://code.jquery.com/mobile/latest/jquery.mobile.min.js"></script>
  <script type="text/javascript" src="js/jquery.audioeffects.js"></script>

Adding the Audio files to the page - prefferably towards the bottom of the page:

Add the following code to the body of the html page 

 <audio id="click1">
 <source src="audio/click1.wav"  />
 <source src="audio/click1.mp3"  />
 <source src="audio/click1.ogg"  />
 <source src="audio/click1.m4a"  />
 </audio>
Give each audio tag a unique 'id' attribute. This 'id' attribute will be selected within the jQuery code to choose the audio effect to play. Each Audio tag should have the source entries for each browser compatibility (.wav, .mp3, .ogg, .m4a); This will ensure that the effect will play in all browsers.

Adding jQuery code:

Add the jQuery code inside a $(document).ready(function(){}); block like so : 

 $(docuemnt).ready(function(){
  $('#divOne').audioeffects({

            theTrigger : "mouseover",
            theStop : "mouseleave"
            audioId : "button1",
            volume: 0.6

    });

  $('#exampleTwo').audioeffects({
            theTrigger: "keydown",
            audioId: "beep3",
            volume: 1,
            keycode: 38,
            callback: function(){alert('this triggered an audio effect when I pressed the "up" arrow key.')}
  })
  });

Options Explained:

  • theTrigger

    - this is the event that triggers the audio effect can be most jQuery events. ('click', 'mouseover', 'mouseout', 'load', 'dblclick','swipe', 'tap', 'scroll', 'keydown', 'keyup', 'focus', 'blur' etc).

  • theStop

    - this is the event that triggers the audio effect to Pause. ('click', 'mouseover', 'mouseout', 'load', 'dblclick','swipe', 'tap', 'scroll', 'keydown', 'keyup', 'focus', 'blur' etc).

  • audioId

    - this is the unique 'id' attribute you set for the 'audio' tag .

  • volume

    - the volume of the audio effect. From 0.0 to 1

  • keycode

    - this is the keycode if using 'keydown' or 'keyup' event - the example 38 is the up arrow key for a complete list of keycodes visit: http://www.k68.org/wp-content/uploads/2010/11/Key_Codes.htm

  • callback

    - add a callback function that will run when event is triggered .

Mobile Events

You can use jQuery-Mobile events such as "tap" and "taphold" but I suggest using a 3rd party plugin such as jQuery.touchSwipe download here: http://labs.skinkers.com/touchSwipe/ for all your 'swipe' events

Please note that not all mobile browsers can handle html5 audio.
